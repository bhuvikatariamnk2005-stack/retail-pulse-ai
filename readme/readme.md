# 📊 Retail Pulse AI Dashboard

## 🚀 Project Overview
Retail Pulse AI is an interactive analytics dashboard built using Streamlit. It analyzes retail sales data to generate insights on sales performance, profit trends, customer segmentation, inventory optimization, and sales forecasting.


## 🎯 Objectives
- Analyze retail sales performance
- Identify top products, cities, and regions
- Perform customer segmentation using clustering
- Predict future sales using time-series forecasting
- Provide interactive dashboard for business insights


## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Plotly, Matplotlib, Seaborn
- Streamlit
- Scikit-learn
- XGBoost
- Prophet
- ReportLab

## 📊 Features
- Interactive filters (Region & Category)
- KPI metrics dashboard
- Sales & Profit analysis
- Customer segmentation (K-Means)
- Churn prediction model
- Sales forecasting (Prophet)
- PDF report generation
- Data download option


## 📈 Key Insights
- Top-performing regions contribute majority revenue
- Few sub-categories dominate sales
- Discounts strongly affect profitability
- High-value customers drive revenue
- Sales show seasonal trends
